1. Create Django Project and application
   - django-admin startproject reactdjango
   - django-admin startapp backend


2. Create React JS application (npm install react)
  - npx create-react-app frontend


3. To install packages In React JS use (npm install)
   - npm install --save bootstrap
   - npm install --save reactstrap react react-dom


NOTE : npm install -g <package> vs npm install <package>
